# Contributor Code of Conduct

kubetidy adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be welcoming, assume good faith, and help make this a community
where everyone can contribute. Harassment and exclusionary behavior are not tolerated.

Report concerns to the maintainers via a private channel listed in the repository, or by
emailing the maintainer team. Reports are handled confidentially.
